# Contributing to Keel

Welcome! Contributions and issues are really appreciated.

If you found a bug, please create an issue.

If you have an idea for a new feature, design change, or anything that modifies the core logic, please first create an issue to discuss it.

For any other code changes that don't modify the core logic (typos, docs, performance improvements), feel free to open a PR!

Please note that I have the final say on what gets merged.

**AI USE**: Using AI is fine as long as you've actually thoroughly read and understand the part of the codebase you're modifying.

Before opening a PR, please make sure:
- all tests pass
- clippy isn't *too* mad
- your code is as fast as possible, then simplified as much as possible without hurting performance
